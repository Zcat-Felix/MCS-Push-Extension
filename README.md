# 美云销图片下载器 - Edge浏览器扩展

## 扩展说明
这是一个专为美云销商品页面设计的图片下载器扩展，支持自动识别和下载商品主图、副图、商品详情图，并自动获取高清原图。

## 功能特性
- ✅ 智能识别美云销商品页面的三种图片类型：主图、副图、商品详情图
- ✅ 自动去除阿里云OSS压缩参数，获取高清原图
- ✅ 支持新旧美云销域名：sales-expedite-ga.midea.com 和 dsdcp.smartmidea.net
- ✅ 一键下载所有图片，支持并发下载
- ✅ 按图片类型自动分类保存到不同文件夹
- ✅ 支持自定义设置：高清原图、分类文件夹、进度显示
- ✅ 提供快捷键支持：Ctrl+S 扫描，Ctrl+D 下载
- ✅ 美观的用户界面，实时显示扫描结果

## 安装方法

### 方法一：从Edge加载已解压的扩展
1. 打开Edge浏览器，在地址栏输入：`edge://extensions/`
2. 开启右上角的"开发人员模式"开关
3. 点击左上角的"加载已解压的扩展"按钮
4. 选择本扩展的文件夹路径（包含manifest.json的文件夹）
5. 扩展将自动安装并显示在扩展列表中

### 方法二：打包安装
1. 在Edge扩展页面开启"开发人员模式"
2. 点击"打包扩展"按钮
3. 选择扩展文件夹路径
4. 点击"打包扩展"生成.crx文件
5. 将.crx文件拖拽到扩展页面进行安装

## 使用指南

### 基本使用
1. 访问美云销商品页面（支持新旧域名）
2. 点击Edge右上角的扩展图标，找到"美云销图片下载器"
3. 点击扩展图标打开控制面板
4. 点击"扫描页面图片"按钮
5. 查看扫描结果（主图、副图、详情图数量）
6. 点击"下载所有图片"按钮开始下载

### 设置选项
- **自动获取高清原图**：自动去除OSS压缩参数，获取原始高清图片
- **按类型分类文件夹**：将图片按主图、副图、详情图分类保存到不同文件夹
- **显示下载进度**：在下载过程中显示进度条

### 快捷键
- `Ctrl + S`：扫描页面图片
- `Ctrl + D`：下载所有图片（需要先扫描）
- `Esc`：关闭设置面板

## 技术实现

### 图片识别策略
1. **CSS类名识别**：通过Img_li、borderHover、md-image等CSS类名识别图片类型
2. **DOM结构分析**：分析图片在页面DOM中的位置和结构
3. **URL特征匹配**：通过URL中的OSS参数识别详情图
4. **尺寸判断**：通过图片尺寸判断主图和副图

### 图片分类
- **主图**：商品主要展示图，通常较大且位于显眼位置
- **副图**：商品轮播缩略图，通常较小且有多张
- **详情图**：商品详情中的图片，通常带有OSS压缩参数

### 高清原图处理
- 自动识别URL中的OSS处理参数（如?x-oss-process=...）
- 去除参数获取原始图片URL
- 支持多种OSS参数格式
