# 安装指南

## 快速安装步骤

### 方法一：Edge浏览器加载已解压的扩展（推荐）
1. **打开Edge扩展管理页面**
   - 在Edge地址栏输入：`edge://extensions/`
   - 或点击菜单 → 扩展 → 管理扩展

2. **开启开发者模式**
   - 在页面右上角找到"开发人员模式"开关
   - 点击开关，将其变为蓝色（开启状态）

3. **加载扩展**
   - 点击左上角的"加载已解压的扩展"按钮
   - 在弹出的文件选择器中，导航到本扩展文件夹
   - 选择包含 `manifest.json` 文件的文件夹
   - 点击"选择文件夹"

4. **完成安装**
   - 扩展将自动出现在扩展列表中
   - 右上角工具栏会出现扩展图标

### 方法二：打包安装
1. **打包扩展**
   - 在Edge扩展页面开启"开发人员模式"
   - 点击"打包扩展"按钮
   - 选择扩展文件夹路径
   - 点击"打包扩展"生成 `.crx` 文件

2. **安装打包文件**
   - 将生成的 `.crx` 文件拖拽到扩展页面
   - 或点击"加载已解压的扩展"选择 `.crx` 文件

## 验证安装

### 检查扩展状态
1. 访问Edge扩展页面 (`edge://extensions/`)
2. 在扩展列表中查找"美云销图片下载器"
3. 确保扩展已启用（开关为蓝色）

### 测试扩展功能
1. 访问美云销商品页面，例如：
   - `https://sales-expedite-ga.midea.com/...`
   - `https://dsdcp.smartmidea.net/...`

2. 点击Edge右上角的扩展图标
3. 应该能看到扩展的控制面板
4. 点击"扫描页面图片"测试功能

## 故障排除

### 常见问题

**问题1：扩展图标不显示**
- 检查扩展是否已启用
- 尝试重启Edge浏览器
- 在扩展页面点击"刷新"按钮

**问题2：扫描失败**
- 确保当前页面是美云销商品页面
- 等待页面完全加载后再扫描
- 检查控制台错误信息（F12 → Console）

**问题3：下载失败**
- 检查下载权限设置
- 确保有足够的磁盘空间
- 检查网络连接

**问题4：图片识别不准确**
- 确保页面是美云销标准商品页面
- 尝试重新扫描
- 检查页面结构是否变化

### 开发者工具调试
1. 按 `F12` 打开开发者工具
2. 切换到"控制台"标签页
3. 查看扩展输出的日志信息
4. 检查是否有错误信息

## 更新扩展

### 手动更新
1. 在扩展页面找到"美云销图片下载器"
2. 点击"重新加载"按钮
3. 或卸载后重新安装

### 自动更新
- 如果通过Edge商店安装，会自动更新
- 手动安装的扩展需要手动更新

## 卸载扩展

### 临时禁用
- 在扩展页面关闭扩展开关

### 完全卸载
1. 在扩展页面找到"美云销图片下载器"
2. 点击"移除"按钮
3. 确认删除

## 系统要求
- ✅ Edge浏览器 88+
- ✅ Windows 10/11, macOS, Linux
- ✅ 网络连接（访问美云销网站）
- ✅ 足够的磁盘空间

2. 检查控制台错误信息
3. 确保使用最新版本
4. 尝试重新安装扩展
